
Это простая программа для сохранения бэкапов на ваш MailRu Cloud


Как с ней работать?
1. Установить зависимости
2. Скомпилировать программу -> pyinstaller backuper.py --onefile --name Backuper --icon="icon.ico"
3. В месте расположения программы создать файл settings.json
4. Заполнить settings.json по примеру из settings.example.json


Пароль для "MAILRU_WEBDAV3_PASSWORD"
создавать здесь -> https://account.mail.ru/user/2-step-auth/passwords



Если всё сделано правильно, то при запуске программы в местонахождении программы начнется:
1. Создание бэкапа
2. Отправка его в директорию "MailRuDirName" указанную в settings.json
2.1 Если такой директории там нет - будет создана
3. Если выскочит ошибка - будет выведена в консоль
4. В итоге бэкап в .zip формате окажется на MailRu Cloud указанной почты.
